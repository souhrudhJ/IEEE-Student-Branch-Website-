# IEEE Student Branch – STM Kannur

Official website for the IEEE Student Branch at St. Thomas College of Engineering and Technology, Kannur.

## Tech stack

- **React** + **Vite**
- **Tailwind CSS** (v4)
- **Framer Motion** (animations)
- **React Icons**

## Setup

```bash
git clone https://github.com/souhrudhJ/IEEE-Student-Branch-Website-.git
cd IEEE-Student-Branch-Website-
npm install
```

## Scripts

| Command        | Description              |
|----------------|--------------------------|
| `npm run dev`  | Start dev server         |
| `npm run build`| Production build        |
| `npm run preview` | Preview production build |

## Sections

Home, About IEEE, About SB, Team (EXCOM), Events, Gallery, Achievements, Membership, Contact.
